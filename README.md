# Sabor Urbano 

## Descripción

Landing Page interactiva creada para un restaurante de comida artesanal.
Permite visualizar el menú, conocer el negocio y contactar mediante un formulario.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Características

- Diseño responsive.
- Navbar fija.
- Tarjetas con CSS Grid.
- Validación de formulario.
